For two strings s and t, we say "t divides s" if and only if s = t + t + t + ... + t + t (i.e., t is concatenated with itself one or more times).

Given two strings str1 and str2, return the largest string x such that x divides both str1 and str2.

 

Example 1:

Input: str1 = "ABCABC", str2 = "ABC"

Output: "ABC"

Example 2:

Input: str1 = "ABABAB", str2 = "ABAB"

Output: "AB"

Example 3:

Input: str1 = "LEET", str2 = "CODE"

Output: ""

Example 4:

Input: str1 = "AAAAAB", str2 = "AAA"

Output: ""​​​​​​​

Constraints:
1 <= str1.length, str2.length <= 1000
str1 and str2 consist of English uppercase letters.




Solution(#1) => T.c = O(n+m) n=len of str1, m = len of str2

Intution =>
1. check if both sring on concatenation are equal or not, if they are divisible strings whose gcd can be calculated then they must be equal on concatenation
2. find gcd of length of both string, and the string starting from pos=0 of length gcd will be the answer

class Solution {
    private:
    int gcd(int dividend,int divisor){
        while(dividend%divisor != 0){
            int rem = dividend%divisor;
            dividend= divisor;
            divisor = rem; 
        }
        return divisor;
    }
public:
    string gcdOfStrings(string str1, string str2) {
        
        if(str1+str2 != str2+str1){
            return "";
        }
        int gcdval = gcd(str1.size(),str2.size());

        return str1.substr(0,gcdval);
        
    }
};
